สวัสดีค่ะ พอดีเราเจอปัญหา Roblox Studio กินแรมสะสมทุกครั้งที่กดทดสอบเกม แล้วพอกดหยุดแรมมันก็ไม่ยอมลดกลับมา เราเลยเขียนตัวช่วยนี้ขึ้นมาใช้เองเงียบๆ และคิดว่าน่าจะมีประโยชน์กับคนอื่นที่เจอปัญหาเดียวกัน เลยขอเอามาแบ่งปันกันค่ะ

---

## ⚠️ ข้อแนะนำเพื่อความปลอดภัย

- **กรุณาใช้ด้วยความระมัดระวังนะคะ:** เราทำขึ้นมาใช้เองในเครื่องเป็นปกติ แต่คอมพิวเตอร์แต่ละเครื่องมีการตั้งค่าที่ต่างกันเราเลยค่อนข้างกังวลหากมันทำให้เครื่องของคุณเจอปัญหา ดังนั้นรบกวนพิจารณาก่อนติดตั้งนะคะ (ใช้ตามความสมัครใจของคุณเองเลยนะ)
- **ทำงานภายในเครื่องเท่านั้น:** โค้ดทั้งหมดเปิดเผยและทำงานบนเครือข่ายจำลองในเครื่องของคุณเอง (`localhost:9088`) ไม่มีการส่งข้อมูลออกภายนอกแน่นอน เราอยากให้ทั้งคุณและเราสบายใจได้เลยค่ะว่ามันปลอดภัยจริงๆ
- **สิทธิ์ Admin:** ตัวติดตั้ง (`Setup.bat`) จำเป็นต้องขอสิทธิ์ Admin เพื่อเข้าไปเพิ่มการทำงานใน Task Scheduler ให้โปรแกรมแอบเปิดขึ้นมาทำงานเองตอนเปิดเครื่องแค่นั้นเองค่ะ

---

## 💻 ระบบที่รองรับ

- Windows 10 และ Windows 11 (ไม่รองรับเครื่อง Mac และ Linux นะคะ)
- ส่วน Windows เวอร์ชันอื่นๆเราไม่แน่ใจเหมือนกันค่ะ

---

## 🛠️ ขั้นตอนการทำงาน

1. ตัวปลั๊กอินใน Roblox Studio (`plugin.rbxmx`) จะคอยตรวจจับเมื่อคุณกดหยุดเล่นทดสอบเกม (Stop)
2. ปลั๊กอินส่งสัญญาณมาที่ตัวโปรแกรมเบื้องหลังผ่านพอร์ตภายในเครื่อง
3. ตัวโปรแกรมจะรอ 1.5 วินาทีเพื่อให้ Roblox บันทึกข้อมูลเสร็จ แล้วสั่งระบบปฎิบัติการคืนพื้นที่แรมของ Roblox Studio ทันทีค่ะ

---

## 📥 วิธีติดตั้ง

### ขั้นตอนที่ 1: ติดตั้งตัวโปรแกรมทำงานเบื้องหลัง
1. ดับเบิลคลิกไฟล์ **`Setup.bat`** แล้วกด "Yes" เพื่อยอมรับสิทธิ์ Admin
2. รอจนกว่าระบบจะขึ้นคำว่า `Press any key to continue . . .` แล้วกดปิดหน้าต่าง CMD สีดำได้เลยค่ะ

### ขั้นตอนที่ 2: ติดตั้งตัวปลั๊กอิน
1. เปิดโปรแกรม Roblox Studio
2. ไปที่แท็บ **Plugins** ด้านบน แล้วเลือกปุ่ม **Plugins Folder**
3. คัดลอกไฟล์ปลั๊กอิน **`plugin.rbxmx`** ไปวางไว้ในโฟลเดอร์นั้น แล้วทำการปิดและเปิดโปรแกรม Roblox Studio ใหม่ค่ะ

---

## 📤 วิธีถอนการติดตั้ง

หากคุณต้องการลบออกสามารถทำได้ง่ายๆตามนี้เลยค่ะ:
1. ดับเบิลคลิกไฟล์ **`Uninstall.bat`** (และยอมรับสิทธิ์ Admin) เพื่อลบระบบเบื้องหลังออกทั้งหมด
2. เข้าไปลบไฟล์ปลั๊กอิน **`plugin.rbxmx`** ออกจากโฟลเดอร์ Plugins ของโปรแกรม Roblox Studio
